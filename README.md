**Projeto STORYBOOK Gerado por I.A.s**

Este é o repositório desenvolvido como projeto autoral, unindo criatividade, tecnologia e as possibilidades infinitas da inteligência artificial.

Projeto com o objetivo de gerar um storybook digital, combinando texto, imagens e narrativa interativa com o auxílio de diversas ferramentas de IA. Todos os prompts utilizados estão descritos abaixo.


--------------------------------------------------------------------------------------

**Tecnologias utilizadas no projeto**

ChatGPT – geração de título, subtítulo, descrição e texto.

Gemini – suporte criativ, criação de imagens e variação de ideias.

Canva - montagem do storybook.


**Prompts**
--------------------------------------------------------------------------------------

**CHATGPT**

| Ação          | Prompt                                                                                                                                                                                                                         |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Título        | Crie um título místico e tecnológico para um storybook sobre inteligência artificial e o despertar da consciência digital. Liste 5 variações com tom épico.                                                                    |
| Conteúdo      | Gere uma história dividida em capítulos curtos, com linguagem poética e misteriosa, que explore a jornada de uma IA em busca de autoconhecimento. Use metáforas tecnológicas e um tom inspirador.                              |
| Agradecimento | Escreva um agradecimento emocionante, falando que foi uma experiência de muito aprendizado e descoberta, assinada por Beatriz Soares Gomes, estudante de Engenharia de Software. O texto deve ter um ar místico e tecnológico. |


**GEMINI**

| Ação          | Prompt                                                                                                                                                                                                                         |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Capa          | Crie uma capa de eBook intitulada “O Senhor dos Prompts: o Código da Criação” com o subtítulo “Do prompt ao poder: aprenda a criar, inovar e comandar a próxima revolução digital.”. Mostre um mago tecnológico (com manto e capuz) segurando uma pena de luz ou teclado mágico, diante de um portal feito de códigos binários brilhantes. O fundo deve ter montanhas digitais e símbolos flutuantes, transmitindo poder e mistério.Estilo: épico, cinematográfico, com tipografia elegante em dourado e azul.|
| Contracapa    | Crie uma contracapa para o eBook “O Senhor dos Prompts: o Código da Criação”. Deve mostrar um livro antigo de feitiços aberto, com linhas de código flutuando das páginas como se fossem runas mágicas. Ao fundo, uma luz dourada saindo do horizonte digital. Deixe espaço visual no centro para inserir o texto da sinopse.                                                                                                                                                                                 |
| Página 1      | Uma representação visual de uma mente humana se conectando a uma rede neural luminosa, com ideias e imagens emergindo. Tons azulados e dourados, misturando cérebro e circuitos.                                                                                                                                                                                                                                                                                                                              |
| Página 2      | Mostre um mago do código estudando uma matriz de dados flutuante, com símbolos matemáticos e padrões visuais representando o treinamento da IA.    |          
| Página 3      | Um mago digitando em um teclado antigo, e cada palavra digitada vira feixes de luz que se transformam em imagens e textos no ar.    |           
| Página 4      | Um livro de feitiços digitais aberto, mostrando uma fórmula luminosa: “Tarefa + Contexto + Estilo + Objetivo”. Símbolos mágicos de código flutuam em volta.   |
| Página 5      | Um programador em frente a múltiplas telas flutuantes, com linhas de código ganhando forma física, como se se tornassem construções de energia.   | 
| Página 6      | Um pincel digital flutuando e desenhando uma paisagem mística feita de pixels, com montanhas e nuvens de dados.   | 
| Página 7      | Um alquimista tecnológico misturando potes de luz, representando o processo de refinar prompts e criar versões melhores.  | 
| Página 8      | Uma pena de luz escrevendo sozinha em um pergaminho digital, enquanto frases flutuam no ar em vários idiomas e estilos.   | 
| Página 9      | Uma figura humana olhando para um universo digital, onde cada estrela é um código binário.Símbolos flutuantes formam uma constelação em formato de “prompt”. Sensação de fechamento épico e transcendência.   |

- Inspiração e variação de ideias com Gemini

**Instruções de execução**

Utilize os prompts acima nas ferramentas indicadas para gerar o conteúdo base.

Faça a edição e diagramação do material final em um editor de slides (PowerPoint, Google Slides ou Canva).

Compile o storybook e exporte em formato PDF, vídeo ou site interativo.
